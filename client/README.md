# 🛒 GroceShop - Client Application (Frontend)

Ini adalah direktori frontend aplikasi **GroceShop** yang dibangun menggunakan React, TypeScript, Vite, dan Tailwind CSS v4.

Untuk melihat peta jalan lengkap proyek (Roadmap), struktur data model, panduan sistem desain (warna & font kustom), serta dokumentasi keseluruhan, silakan merujuk ke berkas dokumentasi utama di root direktori proyek:

👉 **[Buka Utama README.md di Root](../README.md)**

---

## ⚡ Quick Start

1. Pastikan Anda berada di direktori `client`:
   ```bash
   cd client
   ```
2. Jalankan server lokal:
   ```bash
   npm run dev
   ```

## 🛠️ Stack & Library yang Digunakan di Sini

* **Core & Build Tools:** React 19, TypeScript, Vite, Tailwind CSS v4 (with `@tailwindcss/vite`).
* **Routing & UI Utils:** `react-router-dom`, `react-hot-toast`, `lucide-react`, `@icons-pack/react-simple-icons`.
* **Maps & Geolocation:** `leaflet`, `react-leaflet`, `@types/leaflet`.
